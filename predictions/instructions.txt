Παιδιά όπως σας είπα κι από κοντά, ανέβηκαν στο github τα δεδομένα για την αξιολόγηση του ερωτήματος 3 της εργασίας. 
Τα predictions για τα δεδομένα αυτά θα πρέπει να τα ανεβάσετε πάλι στο github στη διαδρομή predictions/test_preds.csv 
Τον φάκελο predictions θα πρέπει να τον δημιουργήσετε εσείς. 
Το αρχείο test_preds.csv θα πρέπει να είναι ένα csv με πρώτη στήλη τα user_id του test set και δεύτερη τις προβλέψεις σας για το πλάνο του αντίστοιχου χρήστη (1 για paid, 0 για standard). 
Πρέπει να υπάρχουν όλα τα user_id που εμφανίζονται στο test set και ένα prediction ανά χρήστη. 
Το αρχείο αυτό θα πρέπει να το έχετε έτοιμο μέχρι την Τρίτη το πρωί όπου θα το πάρουμε και θα το αξιολογήσουμε βάσει το macro-averaged f1 score των 2 κλάσεων.
